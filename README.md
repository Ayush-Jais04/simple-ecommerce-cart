# simple-ecommerce-cart

## 🛒 Features

### ✔ Product Catalog  
- Shows a list of products with IDs, names, and prices  
- Lets the user select items by entering product IDs  

### ✔ Shopping Cart  
- Add items to the cart  
- Remove items  
- Update quantity  
- View cart with total price  
- Clear the entire cart  

### ✔ Checkout System  
- Displays the total before checkout  
- Asks for confirmation  
- Clears cart after successful checkout
